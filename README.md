# koko-contextos-agents 🤖

Это open-source набор навыков (skills) и правил поведения для ИИ-ассистентов. Пакет автоматически устанавливает папку `.agents` в ваш проект, обучая вашего ИИ-помощника лучшим практикам разработки (UI Design, Архитектура, Безопасность и многое другое).

## 🚀 Установка

Вам не нужно ничего клонировать вручную. Просто откройте терминал в корне вашего проекта и выполните:

```bash
npx koko-contextos-agents
```

Скрипт автоматически создаст папку `.agents` и скопирует все необходимые инструкции.

## 🛠 Что внутри?
- **Правила проекта (AGENTS.md):** Базовые инструкции, которым ИИ должен следовать безусловно.
- **Навыки (Skills):** Готовые промпты и руководства по конкретным технологиям (React, Next.js, FastAPI, Node.js).
- **Паттерны проектирования:** Архитектурные правила, DDD, микросервисы.
